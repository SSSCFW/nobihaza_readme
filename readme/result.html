クリアリザルト算出法


クリアリザルトは二周目難易度でないと出せません。
デフォルトのEasyEdgeやHardEdgeでは見れないです。


＜クリアリザルト算出法：のび太編＞
高得点を目指す方向けの資料です。

【クリアボーナス】
HardEdge：１０００点
熱血：１００００点
オワタ：３００００点

重複するので、「おわた」をクリアするとクリア持ち点は４１０００になります。
持ち点的には圧倒的に有利ですが、セーブペナルティのかさむ可能性の高い「おわた」では、あっという間に持ち点がなくなります。


【回復アイテムボーナス】
止血剤：５点
グリーンハーブ：５点
ブルーハーブ：５点
調合したハーブ：１００点
救急スプレー：１０点
栄養ドリンク：５点
心眼の書：５００点
体力増強剤：１０００点
心力増強剤：１０００点
すっぽんぱわー：１０００点

EasyEdgeやHardEdge相当難易度で救急スプレーを９９個にしても、熱血のすっぽんぱわー１個でひっくり返されます。
高得点を狙うなら熱血以上でプレイするよりありません。
ただし、すっぽんぱわーは金田からしか買えません。
そのため、タイラント戦までに金になるものを売り払わないといけないことになります（熱血では「守銭奴の魂」はない）。
体力増強剤と心力増強剤は使わずとっておけば、ポイントになる寸法です。


【クエストボーナス】
スコア条件に指定された資料１つにつき５００点です。
また、アームパーツの正規入手も５００点となっています。
なお、「プレイングマニュアル」と「新訳版について」はスコア条件に含まれていません。
（注：正確にはアームパーツ入手ではなく、入手条件を満たしてカプセル部屋に行くこと。つまり、取る必要はないです。まあ、ワープが使えるので、普通は取るでしょうけど）


【「消火器を壊せ」ボーナス】
マップ中に配置された消火器を破壊すると、１つにつき５００点です。
消火器は全部で３０個あります。
消火器の場所については攻略ページで確認してください。


【魔空空間ボーナス】
小型チェーンソー・テンプレート・ゴールドカードを所持していることによるペナルティが免除されます。
「え？　それだけ？」
と、思うかもしれませんがゴールドカードで弾を入手し、その上で金田と取り引きすると……。
どれだけ少ないセーブで魔空空間を切り抜けられるかが勝負です。
すっぽんぱわー大量購入のチャンス。


【なめてるやつで賞】
最終戦の最中にパンツもぎとっているともらえます。
１５８２点（イチゴパンツ）


【強化ペナルティ】
EasyEdgeの死神のドーピングアイテムを使うと、このペナルティに引っかかります。
具体的には装備品なしの状態でＨＰ１５０以上、ＴＰ６０以上、攻撃力５以上、防御力５以上です。
死神のドーピングアイテムを使わなければ、これ以上のステータスにはなりません。
各ステータスごとに１００００点引かれるので、最大４００００点引かれます。


【アイテムペナルティ】
ゴールドカード：－１００００点（※魔空空間クリアで免除）
テンプレート：－１００００点（※魔空空間クリアで免除）
小型チェーンソー：－１００００点（※魔空空間クリアで免除）
サイコガン：－２００００点

一部の特殊難易度でプレイすると、デフォルトで引かれることになります。
チキンナイフ等は引かれませんが、「ねっけつ」以上では使えないのでポイントが頭打ちします。


【チートペナルティ】
チキンナイフ一振り：－５０００点
Ｓショットガン改１発：－２５００点
暴れん棒将軍一振り：－５００点
水鉄砲１発：－５００点

所持自体は引かれませんが、使うとポイント引かれます。
なお、振らなければいいので、チキンナイフを高速移動に使うのは問題ありません。


【ポンコツペナルティ】
ポンコツ度（ＰＱ）が２以上だと、ＰＱ×１０００点引かれます。


【セーブペナルティ】
セーブ１回につき１００点引かれます。
プレイ時間の方は評価対象にしていないので、極限プレイに臨む人はつけっぱで中座してください。
最終戦より後のセーブはこのペナルティの対象にはなりません。
戦果の保存をしたい場合はエピローグの電話でセーブしておくといいと思います。









＜クリアリザルト算出法：マ・クベ編＞
以下の項目を足したものがポイントとなります。

【弾数ボーナス】
小型ミサイル弾：５００点
小型浮遊機雷弾：１２００点

小型ミサイルは３０発撃つまで１つ消費されません。
撃ったら確実に一つ減る機雷より融通が利きます。


【クエストボーナス】
白磁の壷一つにつき１０００点。
壷は合計５つで、隠し場所は以下の通り。

１．目覚めの給食室（置いてある）
２．犬檻前（設置物破壊）
３．犬檻内（置いてある）
４．パソコン（２回以上セーブしなければ破壊可能）
５．ハンター通路（フロア内に隠し通路あり）


【パロメーターボーナス】
ＨＰ１につき１００点。
ＴＰ１につき２５０点。

突撃一回で１０００点損する計算になります。
