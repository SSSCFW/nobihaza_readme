★銃剣システムを使いこなせ！
新訳のびハザでは、メニュー画面を開くことなく銃器と格闘武器を同時に使えます。
隙は大きいけれど遠距離では絶対の安心感を誇る銃器。

ふたつを間合いによって使い分けると、戦術の幅は広がります。
