登場キャラクター



野比のび太
出典：ドラえもん		 本作の主人公。
趣味は昼寝で、射撃とあやとりが得意なこと以外はまるでダメな少年。
そんなのび太が、今回はバイオハザードという事件に巻き込まれることになる。
一人では到底サバイバルホラーなど生き残れそうにない彼だが、彼には頼れる仲間と、そして勇気があった。

ドラえもん
出典：ドラえもん		 あまりにダメすぎるのび太の未来を変えるべく、２２世紀から２０世紀に派遣されたお手伝いロボット。
普段はポケットから様々な未来の道具を出してのび太を助けてくれるが、肝心な時には役に立たないことが多い。
一応猫型ロボット。
しかし、耳をネズミにかじられてしまい青いタヌキのような外見になってしまっている。
また、セワシ（のび太の孫）に購入された時から不良品で、様々な機能が故障中である。

源静香
出典：ドラえもん		 のび太が恋している女の子。
実際はお転婆な面もあるが、のび太の脳内では「やさしくておしとやかな女の子」と、かなり美化されている。
恋は盲目とはよく言ったものである。
でも、今回はお転婆にしてもちょっと度が過ぎるような……。
気のせいかな？

ジャイアン
出典：ドラえもん		 本名は剛田たけし。
暴君を絵に描いたようなガキ大将で、のび太を筆頭に、理由もなく殴る蹴るの暴行を受けた被害者は数知れない。
しかし、ガキ大将とはただの乱暴者では務まらない。
今回のような事件の際には、勇猛果敢に率先して事に当たる頼れるリーダーでもあるのだ。
自分勝手な都合で人に命令するのは相変わらずだけどね。

骨川スネ夫
出典：ドラえもん		 金持ちの息子であることを鼻にかける嫌味な少年。
普段から、のび太達同級生に対して自慢をするためのネタを仕込むことに余念がない。
しかし、明日に繋がらない無駄知識の数々も、バイオハザードという異常事態の中では仲間達の大きな助けとなるのだった。
今回は、２０世紀の小学生にはまだ馴染みの薄いコンピューターの知識をフル動員してサポートしてくれる。

出木杉英才　
出典：ドラえもん		 静香が好意を抱いているため、のび太が一方的にライバル視している天才児。
勉強にスポーツ、何をやらせてもできるため、自然と畏怖の対象となって同級生からは距離を置かれがちである。
そんな彼とも分け隔てなく付き合えるのが下心のない静香の優しさのはずなのだが、やきもちを焼くばかりののび太がそれに気付ける日はくるのだろうか？

先生
出典：ドラえもん		 のび太・静香・ジャイアン・スネ夫・出木杉の担任。
避難した学校で出会うことになるが……。

ドラミ
出典：ドラえもん		 耳を失ったドラえもんの悲しみを癒すために作られた、という設定の兄妹機。
そのスペックはドラえもんはおろか、同時代の猫型ロボットのなかでもトップクラスとなっている。
しかし、不良品のドラえもんを買うのがやっとだったはずのセワシ（のび太の孫）が、どうしてこんな高性能ロボットを特注できたのだろうか？
ある意味不思議である。

安雄
出典：ドラえもん		 ジャイアンの取り巻きとして原作で名前が判明している少年の一人。
今回の事件では、学校のある場所で出会うことになる。

はる夫
出典：ドラえもん		 安雄と同様に、ジャイアンの取り巻きとして原作で名前の判明している少年の一人。
学校に避難してからは、のび太達とは別に事態を打開するために動いていたようだが……。

緑川聖奈
出典：のびハザオリジナル		 のび太達の学校の児童会長をしている６年生の少女。
のび太達は５年生なので、１年年上ということになる。
テニス部の部長でもあるようで、割と活動的なようだ。
丁寧な物腰からは育ちのよさが伺える。

片瀬健治
出典：のびハザオリジナル		 近所の高校生。
当初は頼りにならなそうな面々の揃った場所に逃げ込んだことを後悔していたが、のび太達が行動する姿を見て考えを改める。
不良っぽい外見とは裏腹に、つっぱる相手はちゃんと選ぶようで意外に常識人。
ちなみに、本家のびハザでは「翁蛾」という苗字だった。

金田正宗（かねだせいしゅう）
出典：のびハザオリジナル		 のび太の町の町内会長。
そんな実権のあるような役職でもないはずなのに、のび太達に対して極めて横柄な態度を取る。
その横柄さは、どこかの会社の社長でもあるから、という理由では理解しがたいレベルに達している。
気が動転して頭がおかしくなっているようだが、元から頭の少しおかしい人ではあったようだ。

山田太郎
出典：のびハザオリジナル		 のび太達の学校の１年生。
バイオハザードという異常事態を生き延びるには、あまりに非力で幼い存在である。

Ｄｒ．ライト
出典：ロックマン		 不思議なカプセルから、のび太を助けるような言動を行う謎の人物。
のび太に何か特別な感情を持っているようだが、のび太にはまったく覚えがないようだ。
作中においては、２４世紀か２５世紀（と推定される時代）の人物であると思われる。

ゼロ
出典：ロックマンＸ		 ２６世紀（と推定される時代）の戦闘型ロボット。
シグマのような凶悪犯罪を行うロボットを取り締まる任務についている。

原作を知っている人のために付け加えておくと、恐らくはＸ６～Ｘ７の間のゼロと思われる。
つまり、親友のエックスは ひきこもり 主役降板中。

シグマ
出典：ロックマンＸ		 最低の野望と、最低の人格と、最低の容姿を兼ね備えたイレギュラー（狂ったロボットの意味。人間で言えば狂人）の王。
世界から人間を駆逐し、ロボットがそれに取って替わることを目論んでいる。
彼が２２世紀に漂着したことから大惨事が始まった。

トレノフ・Ｙ・タイムスキー
出典：本作オリジナル		 あちこちで見つかる「時空の書」の原作者。
２２世紀のタイムスキー時空研究所の責任者で、時空研究の第一人者でもあるようだが、それが２０世紀のバイオハザードと何の関係があるのかは謎である。
