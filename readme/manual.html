操作方法など


＜操作方法＞
【通常モード時】（ボタンはキーボード）
↑……上に進みます
↓……下に進みます
←……左に進みます
→……右に進みます
Ｚ……話す・調べる・決定
Ｘ……メニュー・キャンセル
Shift押し続け……武器構え状態に移行（武器の装備が必要）
Shift……乗り物に乗る（乗り物の装備が必要）

【武器構え時】（武器構え中は向き固定移動になります）
↑……上に進みます
↓……下に進みます
←……左に進みます
→……右に進みます
Ｚ……メイン武器攻撃
Ｘ……サブ武器攻撃

【乗り物使用中】
↑……上に進みます
↓……下に進みます
←……左に進みます
→……右に進みます
Ｘ……特殊アクション
Shift……乗り物を降りる

【見切り】（上級者テクニック）
敵の攻撃に合わせてタイミングよくＺキーを入力。
成功すると、敵の攻撃を無効化することができます。

【特殊なキー】
Ｆ４……フルスクリーンとウインドウの切り替え
Ｆ５……ウインドウモード時にウインドウサイズを切り替え
Ｆ１２……タイトル画面に戻る（リセット）



＜ゲームパッド＞
自動で割り振られます。
どのキーがどのボタンに対応しているかは、実際に操作して掴んでください。
一つの目安として、のび太を操作可能になったら以下のことを確かめてみましょう。
  • メニュー画面を開けるのがＸキー対応ボタン
  • メニュー画面でメニューの項目を選べるのがＺキー対応ボタン
  • 通常画面で「武器がセットされていない」警告が出るのがshiftキー対応ボタン（武器を装備していれば構え動作になります）

パッドが、ちゃんと接続されているのに反応しないという時は、ＲＰＧツクール２０００に未対応の可能性があります。
そういう方はJoyToKeyを使ってみてください。
これでもダメなら、ちゃんと接続されていないか、ゲームパッドが壊れているかのどちらかです。
