ゲーム中のトラブル
マップ移動中や、宝箱を開けた時とかにフリーズします。
激しく戦闘をした後などにまれに発生します。
原因は、おそらくツクール内部の処理落ち。
構えて銃撃をすれば直ることもありますが、こまめなセーブが一番の対策です。



ロードしたらイベントがおかしくなりました。
セーブアイテム（インカム）や、「不安定」と注意書きされているセーブは、設置パソコンに比べて安定しません。
バグ状態を保存してしまうこともあるので、節目節目で設置パソコンのセーブデータを作るようにしてください。

また、本体を更新したときも、インカムなどのデータは使用不能に陥りやすいです。
上書き更新をかける前は、必ず設置パソコンを使うようにしてください。



構えられなくなってしまいました。　or　構えが解けなくなりました。
仕様上、滅茶苦茶にボタンを連打したりすればイカれることがあります。
意味もなくマップ移動中に構えキーを連打したりするのは、処理落ち原因にもなるのでやめた方がいいです。



構えを解いても向きが固定されたままです。
軽度の処理落ちです。
もう一度構えて解除すれば、たいていは元に戻ります。
攻撃しながらマップ移動、攻撃が終わるのを待たず移動しながら構え解除……などの動作で、時々起こるようです。
仕様のようなものと思ってください。



会話中に「ファイルが開けない」とエラーが出てゲームが終了してしまいました。
処理落ちで顔アイコンの読み込みに失敗したようです。
のびハザは、かなりメモリに負荷がかかっていたりするので、時々そのような不慮の事故が起こります。
こまめにセーブするようにしてください。



敵に組み付かれると、一時的に動作がおかしくなることがあります。
中断された動作が再開されているようです。
その動作が終了すれば、その後は普通に行動できます。



物を調べている間に攻撃されました。その後フリーズしました。
調べている最中に殺されるとフリーズするのは、半ば仕様です。
フリーズとはいえ既にゲームオーバー状態ですので、諦めてＦ１２でタイトルに戻ってください。
敵が近くにいる状態で物を調べたり、場所移動をするのは大変危険です。



「無効なイベントが指定されました」とエラーが出てゲームが終了してしまいました。
イベント中に起きた場合は、プログラムミスの可能性が高いです。
申し訳ありませんが、バグ報告お願いします。
ただ、戦闘中の場合は処理落ちの可能性があります。
特に、多段ヒット系の武器を使用した時、ＰＣとの相性によっては起こりやすくなるようです。
仕様上こればかりはどうしようもないので、ロードして何事もなければ、気にしないようにしてもらえると幸いです。







ゲーム以前のトラブル
どれを押せばゲームが始められるのでしょうか？

↑コレをダブルクリックしてください。



｢ファイル システムは開けません｣と表示されてゲームが始められません。
新訳のびハザはＲＰＧツクール２０００で作られています。
このツールで作られたゲームをプレイするには、ランタイムというものが必要になります。
こちらのものをダウンロードしてインストールしてから、もう一度ゲームを起動しなおしてください。
ちなみに、これも無料ですから安心してください。



さっきのランタイムをインストールしたのにゲームができません。
ダウンロードに失敗したか、そもそもダウンロードしたファイルを展開していない可能性があります。
もう一度こちらからランタイム（2000rtp.exe）をダウンロードして、今度は必ず落としたものをダブルクリックして展開するようにしてください。
インストールが完了すればゲームは起動できるはずです。



ランタイムもちゃんとインストールしたのに起動できません。
お使いのパソコンはWindowsでしょうか？
新訳のびハザは、原則的にWindowsでしか動作しません。
また、自作パソコンなど特殊な環境のパソコンでは、ゲームをするための環境自体が整っていない可能性もあります。
特殊な環境下でのプレイについては、自己責任ということでご容赦願います。

細かいことについては、Ｗｉｋｉのトラブルシューティングも参考にしてみてください。
